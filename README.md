# Tankard — система обработки игровой статистики World of Tanks

[![Python](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Tankard — автоматизированная система анализа статистики «Мир Танков» с выдачей персональных рекомендаций на основе гибридного подхода (статистический анализ + эвристики ИИ). В отличие от существующих агрегаторов (Tomato.gg, встроенная статистика Lesta), Tankard не только показывает цифры, но интерпретирует их и даёт конкретные советы по улучшению игры.

Добовляю ии будет круто 🍺

## Архитектура

Система реализована в виде модульного Python-приложения, состоящего из трёх компонентов:

| Модуль        | Файл              | Назначение                                                                            |
|---------------|-------------------|---------------------------------------------------------------------------------------|
| Парсер        | `parser.py`       | Получение и обработка данных из API Wargaming.net                                     |
| AI-аналитик   | `ai_module.py`    | Выявление слабых сторон, распознавание игровых паттернов, формирование рекомендаций   |
| Интерфейс     | `interface.py`    | Взаимодействие с пользователем, запуск модулей, отображение результата                |
